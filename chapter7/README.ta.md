# அத்தியாயம் 7 · மாதிரி பிந்தைய பயிற்சி

> முன்-பயிற்சி, SFT, RL என்ற மூன்று நிலைகளின் முழுக் கண்ணோட்டம். எப்போது SFT, எப்போது RL எனத் தேர்வு செய்வது, RLHF, வழிமுறை ஒப்பீடுகள், தரவு மற்றும் சூழல், மேலும் மாதிரி கருவி அழைப்பைக் கற்றுக்கொள்ளச் செய்தல் மற்றும் மாதிரி செயல்திறனை (sample efficiency) மேம்படுத்துதல் போன்ற எல்லை ஆராய்ச்சிகள்.
