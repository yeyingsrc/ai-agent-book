# அத்தியாயம் 8 · ஏஜென்ட் சுய-பரிணாமம்

> எடைகளை மாற்றாமலும் வளர முடியும். மூன்று கற்றல் முன்னுதாரணங்கள் — அனுபவத்திலிருந்து கற்றல், "கருவி பயனரிலிருந்து கருவி படைப்பாளராக மாறுதல்" வரை — ஏஜென்ட்டை "புத்திசாலி" என்பதிலிருந்து "திறன் மிக்க" நிலைக்குக் கொண்டு செல்கிறது.
