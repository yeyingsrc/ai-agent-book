# அத்தியாயம் 5 · Coding Agent மற்றும் குறியீடு உருவாக்கம்

> குறியீடு என்பது "புதிய கருவிகளை உருவாக்கக்கூடிய கருவி" ஆகும், பொது-நோக்க ஏஜெண்டின் மேல்-திறன் (meta-capability) ஆகும். உற்பத்தி-தர Coding Agent ஐ எடுத்துக்காட்டாகக் கொண்டு, இந்த மிகச் சக்திவாய்ந்த பொது-நோக்கக் கருவியின் முழுமையான செயலாக்கத்தைக் காட்டுகிறது.
