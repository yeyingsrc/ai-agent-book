# அத்தியாயம் 3 · பயனர் நினைவகம் மற்றும் அறிவுத் தளம்

> ஏஜென்ட் அமர்வுகளுக்குக் குறுக்கே பயனரை நினைவில் வைத்திருக்கவும், வெளிப்புற அறிவை அணுகவும் செய்கிறது. பயனர் நினைவக அமைப்புகள், RAG அடிப்படைக் குழாய்கள், மற்றும் தட்டையான உரையைத் தாண்டிய அறிவு ஒழுங்கமைப்பு மற்றும் மீட்டெடுப்பு (கட்டமைக்கப்பட்ட குறியீடுகள், அறிவு வரைபடங்கள் போன்றவை) ஆகியவற்றை உள்ளடக்கியது.
