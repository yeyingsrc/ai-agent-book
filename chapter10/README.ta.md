# அத்தியாயம் 10 · பல-ஏஜென்ட் கூட்டுச் செயல்பாடு

> குழுவின் நுண்ணறிவு தனிநபரின் நுண்ணறிவை விட உயர்ந்ததாக இருக்கலாம். பல-ஏஜென்ட் வகைப்பாட்டுக் கட்டமைப்பு, எப்போது உண்மையில் ஒற்றை ஏஜென்ட்டை விடச் சிறந்தது, சூழலைப் பகிர்ந்து கொள்ளும் மற்றும் பகிராத ஒத்துழைப்பு, தோல்வி முறைகள், மற்றும் தோன்றும் "ஏஜென்ட் சமூகம்" ஆகியவற்றை விவரிக்கிறது.
