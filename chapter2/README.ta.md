# அத்தியாயம் 2 · சூழல் பொறியியல்

> சூழல் ஏஜெண்டின் திறனின் மேல் வரம்பைத் தீர்மானிக்கிறது. பெரிய மாதிரி API இன் சூழல் அமைப்பு, KV Cache நட்பு வடிவமைப்பு, விரைவு பொறியியல் (prompt engineering), மாறும் விரைவுகள் மற்றும் Agent Skills, நிலைப் பட்டை மேல்தரவு (metadata), மற்றும் சூழல் சுருக்க உத்திகள் ஆகியவற்றை ஆழமாக ஆராய்கிறது.
